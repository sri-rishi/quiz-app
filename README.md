## Features ?

1. Home With Category
2. Rules Page
3. MCQ Based Question 
4. Quiz Result 
5. User Profile 
6. Timer for Questions
10. Authentication
      - User Signup
      - User Login
      - User Logout
11. Full Responsive

  

## **Built with -**

- React with Typescript 
- Firebase for Authentication
- Material UI 
- Vanilla CSS
- React Context API + useReducer
- React Router v6
